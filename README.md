# PWA Track

Prague Watch Archive — internal reselling analytics dashboard.

## Deploy to Vercel (5 min)

1. Go to github.com → click **+** → **New repository**
2. Name it `pwa-track`, keep it public, click **Create repository**
3. Click **uploading an existing file** link on the page
4. Upload ALL files from this zip (keep folder structure intact)
5. Click **Commit changes**
6. Go to vercel.com → **Sign up with GitHub**
7. Click **Add New Project** → select `pwa-track`
8. Leave all settings as default → click **Deploy**
9. Done — you get a live URL in ~60 seconds
